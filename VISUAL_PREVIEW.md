## 🎨 Color Reference Quick Guide

### Primary Palette
- **Orange**: `#ff6b35` - Primary actions
- **Blue**: `#0071e3` - Secondary actions
- **White**: `#ffffff` - Backgrounds
- **Dark**: `#1d1d1f` - Text

### Gradients
- **Hero**: `linear-gradient(135deg, #0071e3 0%, #00a8ff 100%)`
- **Button**: Orange solid `#ff6b35`
- **Titles**: `linear-gradient(90deg, #ff6b35, #ff9f66)`

### Shadows
- **Light**: `0 2px 12px rgba(0,0,0,0.05)`
- **Medium**: `0 8px 25px rgba(0,0,0,0.1)`
- **Strong**: `0 20px 60px rgba(0,0,0,0.2)`

### Interactive States
- **Hover**: Lift up (-8px) + Shadow increase
- **Focus**: Border color change + Glow effect
- **Active**: Scale down slightly

---

## ✨ Animation Effects

### Hover Effects
- **Cards**: `transform: translateY(-8px)` + Enhanced shadow
- **Buttons**: `transform: translateY(-2px)` + Shadow glow
- **Images**: `transform: scale(1.05)` + Shadow

### Transitions
- **Standard**: `all 0.3s ease`
- **Fast**: `all 0.2s ease`
- **Slow**: `all 0.5s ease`

### Focus States
- **Inputs**: Blue glow `0 0 0 3px rgba(0,113,227,0.15)`
- **Buttons**: Blue outline `0 0 0 3px rgba(0,113,227,0.25)`

---

**Design Philosophy**: Clean, modern, and professional - inspired by Apple's minimalist approach with vibrant accent colors for personality! 🎨✨
